title: Setup
---

## Windows

Prerequisites:

- Visual Studio 2019
- Cmake
- git
- Strawberry Perl

Run `build_release.bat` in `x64 Native Tools Command Prompt for VS 2019`

## MacOS

Prerequisites:

- Xcode
- Cmake
- git
- gettext

Run `build_release_macos.sh`

## Linux (Ubuntu / Debian)

- git
- gettext
- `build-essential` package

Run `BuildLinux.sh -udisr`
