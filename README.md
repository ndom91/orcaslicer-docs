# orcaslicer-docs

Demo [Orca Slicer](https://orcaslicer.com) static docs site.

## Getting Started

1. Clone repository

```sh
git clone https://github.com/ndom91/orcaslicer-docs
```

2. Install dependencies

```sh
cd orcaslicer-docs
pnpm install
```

3. Start development server

```sh
pnpm dev
```
